# SmartFlood

SmartFlood is an advanced flood monitoring and disaster awareness system meant to serve up-to-date weather, tides data, interactive maps, and AI-based flood safety recommendations. The aim of SmartFlood is to increase the understanding of flood hazards using trustworthy environmental data and visual representation.

## Features

• Interactive flood map
• Live weather and tide monitoring
• AI-driven flood safety assistant (AquaBot)
• Location-aware flood data
• Flood analytics dashboard
• Terrain-based flood risk analysis (Work in progress)

## Development

The application is currently under total rework, aimed at the architecture update, including modular development, backend security, API protection, updated UI/UX, and realistic flood risk calculation. In further development, it is planned to include elevation data, terrain analysis, drainage conditions, and historical flood data.

## Purpose

SmartFlood is an educational and research project which seeks how geospatial data, environment monitoring, and modern web technologies can be used to improve flood awareness and disaster preparedness.
